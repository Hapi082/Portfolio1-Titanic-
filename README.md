# 🚢 Titanic 生存予測チャレンジ（XGBoostモデル）

本プロジェクトは、[KaggleのTitanicコンペ](https://www.kaggle.com/c/titanic)で乗客の生存率予測を行います。
データ分析力や機械学習スキル向上を目的としています。

### 使用技術など
- Python 3.10.9
- Numpy
- Pandas
- Scikit-learn
- Matplotlib/Seaborn
- XGBoost
